# 📘 Caderno Temático de Estudos com NotebookLM

### Inteligência Artificial como Ferramenta de Aprendizagem Ativa

## 📌 Sobre o Projeto

Este repositório apresenta um **Caderno Temático de Estudos desenvolvido com apoio da Inteligência Artificial utilizando o NotebookLM**, com o objetivo de transformar a IA em uma ferramenta de **aprendizagem ativa, pesquisa e organização do conhecimento**.

O projeto foi desenvolvido como parte de um desafio prático da plataforma **Digital Innovation One**, que incentiva o uso de IA aliado a pensamento crítico, curadoria de conteúdo e documentação do processo de aprendizado.

A proposta consiste em:

* Selecionar fontes confiáveis
* Utilizar o **NotebookLM** para análise e síntese do conteúdo
* Aplicar **engenharia de prompts**
* Documentar todo o processo de aprendizado
* Consolidar um **Miniguia de Estudo**

---

# 🎯 Contexto e Objetivos

## Tema Escolhido

**Inteligência Artificial aplicada ao aprendizado e produtividade**

O tema foi escolhido devido ao crescimento acelerado das ferramentas de IA e ao impacto que elas têm causado na forma como **estudamos, pesquisamos e organizamos conhecimento**.

Ferramentas como **NotebookLM**, **ChatGPT** e **GitHub** estão transformando a forma como profissionais e estudantes aprendem.

---

## Objetivos de Estudo

Este caderno temático possui os seguintes objetivos:

* Entender como a **IA pode apoiar o aprendizado ativo**
* Aprender a utilizar **NotebookLM para pesquisa estruturada**
* Desenvolver habilidades de **engenharia de prompts**
* Criar um **sistema organizado de estudo com IA**
* Documentar o processo de aprendizado de forma técnica

# 🧠 Engenharia de Prompts e "Cicatrizes"

Durante o estudo, foram testadas diversas abordagens de prompts para extrair informações mais precisas do NotebookLM.

Aqui estão alguns exemplos documentados.

---

# 🔎 Prompt 1 — Exploração Inicial

**Prompt utilizado**

```
Explique de forma simples o que é Inteligência Artificial e como ela pode ser utilizada para melhorar o processo de aprendizagem.
```

**Resultado**

A IA foi apresentada como um conjunto de técnicas que permitem que máquinas aprendam padrões e tomem decisões com base em dados.

**Aprendizado**

Esse prompt gerou respostas muito amplas. Foi necessário refiná-lo para obter respostas mais específicas.

---

# 🔎 Prompt 2 — Refinamento

**Prompt utilizado**

```
Com base nos documentos carregados, explique como a Inteligência Artificial pode apoiar métodos de aprendizagem ativa e organização do conhecimento.
```

**Resultado**

A resposta trouxe conceitos mais alinhados com as fontes, incluindo:

* síntese automática de conteúdo
* geração de resumos
* apoio à revisão de estudos
* organização de conhecimento

**Melhoria obtida**

Adicionar **"com base nos documentos carregados"** fez a IA usar as fontes específicas do NotebookLM.

---

# 🔎 Prompt 3 — Análise Crítica

**Prompt utilizado**

```
Quais são as limitações do uso da Inteligência Artificial no processo de aprendizagem?
```

**Resultado**

A IA destacou limitações como:

* Dependência excessiva da ferramenta
* Possível geração de respostas imprecisas
* Falta de pensamento crítico do usuário

**Aprendizado**

Esse prompt ajudou a desenvolver uma **visão crítica sobre a IA**, evitando uso passivo da tecnologia.

# ⚠️ Dificuldades Encontradas (Troubleshooting)

Durante o processo foram identificadas algumas dificuldades:

### 1️⃣ Respostas Genéricas

Alguns prompts retornavam respostas muito amplas.

**Solução**

Tornar os prompts mais específicos e contextualizados.

---

### 2️⃣ Falta de Referência às Fontes

A IA nem sempre citava as fontes carregadas.

**Solução**

Incluir no prompt:

```
Baseie sua resposta apenas nos documentos fornecidos.
```

---

### 3️⃣ Excesso de Informação

Algumas respostas eram muito longas.

**Solução**

Adicionar restrições:

```
Resuma em até 5 tópicos principais.
```

# 📖 Miniguia de Estudo

## 📌 O que é Inteligência Artificial?

Inteligência Artificial é um campo da computação que busca criar sistemas capazes de realizar tarefas que normalmente exigiriam inteligência humana, como:

* aprendizado
* reconhecimento de padrões
* tomada de decisão
* processamento de linguagem

---

# 📚 Resumo Estruturado

### 1️⃣ Inteligência Artificial

Área da ciência da computação dedicada ao desenvolvimento de sistemas inteligentes.

### 2️⃣ Machine Learning

Subcampo da IA que permite que sistemas aprendam padrões a partir de dados.

### 3️⃣ Aprendizagem Ativa

Método de estudo no qual o aluno participa ativamente da construção do conhecimento.

### 4️⃣ Engenharia de Prompts

Processo de criação de instruções claras para obter melhores respostas de modelos de IA.

---

# 📖 Glossário

**Inteligência Artificial (IA)**
Tecnologia que permite que máquinas simulem capacidades cognitivas humanas.

**Machine Learning**
Método que permite que algoritmos aprendam com dados.

**Prompt**
Instrução ou pergunta fornecida a um modelo de IA.

**Prompt Engineering**
Técnica de elaboração estratégica de prompts para melhorar respostas.

**NotebookLM**
Ferramenta baseada em IA desenvolvida para análise e organização de conhecimento a partir de documentos.
